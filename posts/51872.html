python基础教程

Python中的数据类型

常用的数据类型

int 整型
str 字符串型
float 浮点型
布尔型
list 列表型
元祖型
字典型等

使用type()查看变量的数据类型

a = 12343
print(type(a))

python中定义变量

Python中的变量的定义相对于其他编程语言来讲较为灵活，比如定义变量a的值是10，可以用

a = 10

上图可以看出，python在定义变量时毫无限定，通常是赋予变量的值决定了变量的数据类型，并且python中的变量可以被重写，覆盖

a = 10
a = "qe"

图中变量a的值最终变成了字符串”abcde”

注意事项：python中定义变量时，变量名应遵循以下规则：

​ 1.变量名区分大小写

​ 2.变量名只能以字母或下划线开始

​ 3.变量名的定义不能使用系统关键字

例：

# coding:utf-8
a2 = 33
2a = 33  不能以数字开头
dd = 44
True = 44  True是关键字

整型数据 int

Int型也是各种编程语言中最常见的数据类型，一般指整数型数据，包含正整数，负整数等，在java中会根据整数的值来选择使用长整型或短整型，但是在python中统一为整型

整型数据常用的运算方式：

求整

在python2.7中 使用斜杠实现两个整数相除的求整运算例如：

a = 3/10
print(a)

ps :但是在python3.x版本中取消了求整的运算，斜杠变成了真正的除法

求余

使用%实现两个整数相除的求余运算

a = 10%3
print(a)

其他运算

在python中同样可以实现加法，减法等运算

b = 2-1
c =  2+3

字符串数据 str

在python中，string与char类型统称为str型，也就是说，python中省略掉了字符型，将字符串与字符型使用str来表示，python使用单引号或双引号来声明字符串

# coding:utf-8
name = "jj"
name2  = 'ff'

注意事项：在声明字符串时，单引号与双引号可以嵌套使用但不能重复使用，例如：可以在双引号中嵌套使用单引号，或单引号中嵌套使用双引号，但不能重复使用同一种引号

字符串的索引

在数据库以及其他编程语言中，都会大量使用索引，索引的使用优化了字符串的读取速度，提升了字符串的灵活度，在python中使用[]来表示索引，索引默认从0开始

# coding:utf-8
name = "jj"
name2  = 'ff'
ptint(name2[1])  输出name2中索引为1的字符，由于索引是由0开始，1则代表第二个字符

注意事项：当使用的索引超出字符串的索引范围时，会报错

字符串的切片操作

在python中，可以使用索引的范围实现字符串的切片操作，来截取字符串中的某一段,例如下图中的name[1:3]，截取的是字符串中的索引为1和2的片段

# coding:utf-8
name = "jjqweq"
name2  = 'ffasdada'
ptint(name2[1:3])

name[:3]代表从字符串的第一个字符开始读取

# coding:utf-8
name = "jjqweq"
name2  = 'ffasdada'
ptint(name2[:3])

name[:-2]代表以倒序的方式读取索引，截取的是从字符串开始，到倒数第二位的字符串

# coding:utf-8
name = "jjqweq"
name2  = 'ffasdada'
ptint(name2[:-2])

求字符串的长度

在python中可以通过len()方法直接获取字符串的长度

# coding:utf-8
name = "jjqweq"
name2  = 'ffasdada'
ptint(len(name2))

拼接字符串

在python中可以使用”+”拼接字符串，将两个字符串连接起来

# coding:utf-8
name = "jjqweq"
name2  = 'ffasdada'
ptint(name+name2)

修改字符串中的大小写

在python中，大小写是区分开的，针对一个字符串可以使用指定方法修改字符的大小写，其中upper()是将所有字母改为大写，lower()是将所有字母改为小写

title()是将单词的首字母改为大写

# coding:utf-8
name = "jjqweq"
name2  = 'ffasdada'
ptint(name.upper())
print(name.lower())
print(name.title())

将其他数据类型转换为字符串

在python中可以使用str()方法将其他数据类型转换为字符串，例如：将整数转换为字符串

# coding:utf-8
a = 12345
print(str(a))

布尔型 True/False

Python的布尔型一般用于判断循环语句，其中True代表真实的，存在的。False代表虚假的，不存在的。在下文中将会对这两个字符串做详细解释

列表型 list

在Python中，列表型数据的应用范围很广，常用来存储同类型的数据，或者需要以同等方式保存的数据，例如一组用户名，一组账号信息，一组订单信息等

定义列表

在python中定义列表的方式有以下几种：

# coding:utf-8
list1 = [] 定义空列表
list2 = [1,2,3,4] 定义纯数字列表
list3 = ["q","w"."e","r"] 定义纯字符列表
list4 = [1,"q",3,"ert"] 定义数字与字符串的混合类型列表
list5 = [22,["y","w"],5] 定义嵌套列表

上图中可以看出，python中定义列表不需要声明列表的类型，列表中的每个元素可以随意定义，可以是整型，字符串，列表等等，这样一来大大增加了列表的灵活程度

注意事项：在定义列表时要注意，元素与元素之间要有逗号

列表的索引

列表同字符串一样，也是依靠索引来获取列表中的元素，索引默认从0开始排序，所以列表中的第一个元素的索引默认为0，最后一位的索引为-1

list5 = [22,["y","w"],5]
print(list5[0])
print(list5[-1])

列表的切片

通过对列表切片是通过索引获取列表中指定的部分，可以得到字列表，类似于字符串的切片。

ps :相当于将一个完整的列表切割成几部分

list2 = [1,2,3,4,5,6,7]
print(list2[2:4]) 获取索引为2到索引为3的所有元素，并保存为列表
print(list2[:4])  获取索引为0到索引为3的所有元素，并保存为列表
print(list2[2:])  获取索引为2到列表末尾的所有元素，并保存为列表
print(list2[:-2]) 获取从列表开始，到列表倒数第二位的所有元素，并保存为列表

上图为获取子列表的几种形式，其中”:”起到了获取区间的作用

注意事项：对列表进行切片时，中括号区间中的第二位数取不到，例如

List[:3] 只能取到索引为0~2的3个元素

ps：[:3]这种写法相当于省略掉了0，也就是说，列表切片从0开始时，是可以省略0的

修改、添加、删除列表元素

向列表中添加元素有两种方式，第一种在列表末尾添加元素使用append()方法

List2.append(“abc”) 代表向list2列表末尾添加一个字符串元素“abc”

list2 = [1,2,3,4,5]
list2.append("qqwer")
print(list2)

第二种插入的方法是insert(2,”abc”)是在指定位置插入元素，在索引为2的位置插入字符串”abc”

list2 = [1,2,3,4]
list2.insert(2,"qqwer")
print(list2)

修改列表中的元素可以采用直接修改的方式，例如：list2[3] = “def”

ps：Python中修改某个变量或者元素的值时，都可以采用直接修改的方式

list2 = [1,2,3,4]
list2.insert(2,"qqwer")
list2[3] = "rty"
print(list2)

删除列表中的元素使用pop()，pop()默认删除最后一个元素，如果需要删除指定位置，可以使用带索引的删除方式

list2 = [1,2,3,4,5]
list2.pop()   默认删除最后一位
list2.pop(2)   删除列表中的指定位

统计列表长度、重复元素

在python中可以使用len()方法统计列表的长度

list2 = [1,2,3,4,5]
print(len(list2))  得到列表长度

len()方法的使用，列表最后一位的索引默认为len(list)-1，例如：

list2 = [1,2,3,4,5,6,7]
print(list2[len(list2)-1])  len(list)-1  得到最后一位的索引，在通过索引获取最后一位

使用count()统计列表中元素的出现次数

ps：出现次数为0代表列表中没有该元素

list2 = [1,2,3,4,5,6,78,7,7]
print(list2.count(7))  统计7这个元素在列表出现的次数

列表的去重操作

使用set()方法对列表进行去重操作，该操作会返回一个去重后的列表

list2 = [1,2,3,4,5,6,6]
list3 = set(list2)  使用set对列表进行除重
print(list3)

列表的反转操作

可以使用reverse()方法对列表进行反转，reverse()函数是永久改变列表

# coding:utf-8
lk =[12,342,56,46,45]
lk.reverse()
print(lk)  列表反转是永久改变列表

列表的排序操作

可以使用sort()方法对列表进行排序，sort是永久排序，默认从小到大排序

# coding:utf-8
lk = [1,33,22,44,675]
lk.sort()
print(lk)

在sort()方法中加入reverse参数，可以实现列表的从大到小排序

# coding:utf-8
lk = [1,3,4,55,67,78]
lk.sort(reverse=True)
print(lk)
