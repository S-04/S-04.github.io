1.python 语言介绍

1.1 前言

Python 是一种既简单又功能强大的语言，它注重的是如何解决问题而不是编程语言的语法和结构，同时 python 也属于面向对象编程语言的一种，部分语法也继承了 java 中的简洁，同时剔除了较为复杂的语法。

1.2 python的介绍
Python 当前可以分为两个版本 python2.x 以及较新的 python3.x 版本，python 版本的不同所支持的语法也有些许不同，但是大部分功能以及脚本是可以同时在两个版本的 python 中运行的，在文档内容中将会给大家讲解不同版本的语法区别。

1.3 python的安装

推荐安装python3.x

1.4 python 代码编辑器

Python 的编辑器有很多种，常见的有 pycharm,sublime 等，不同编辑器可以实现的功能也不同，pycharm 功能较为强大，可以实现代码集成，调用系统测试框架等，sublime 编写代码较为方便，代码编写风格比较清新。

综上：代码集成推荐使用 pycharm,脚本编写推荐使用 sublime。

4. python 中的 if 语句

4.1 python 中缩进的使用

在 python 中，只使用缩进来区分代码块，相对于其他编程语言，要更为简洁， 但是也正是存在这种灵活性，一些初学者在编写代码时，也常常会因为缩进而报错。

python 中对缩进没有硬性规定，一个空格，或者多个空格都可以使用，为了保持脚本的统一性，常见的缩进是一个 tab 键(制表符)的缩进，也可以用四个空格来代替，在开始学习 python 时一定要养成良好的缩进习惯。

4.2 if 语句的常见形态

If 判断语句，通常用来做条件的判断，所以在任意一种编程语言中都比较重要， 从根本意义上来讲，一个大型软件也是由无数个判断加循环组成的，由此可见判断语句在代码中的地位。在 python 中 if 语句后需要跟判断条件，并要以冒号引出需要执行的语句。

If可以理解为“如果，即如果条件成立，就执行下面的语句

4.3 elif,else 等语句的使用

Python 中使用 if语句时，通常会搭配 elif else 语句， 可以增加判断语句覆盖的范围，if 可以理解为”如果”，那么 elif 理解为“或者如果“，else 可以理解为 “其他”。

注意事项：当代码中同时存在 if，elif，else 时，这三个只能执行其中一个，不存在同时执行的情况

4.4 常见的条件判断符号

在使用 if 语句时通常会有两个条件同时存在的情况，这样就需要对两个条件进行判断，常用的语句有 and 和 or，and 表示并且，两个条件都需要成立，or 表示或者，两个条件成立一个即可

注意事项：在多个条件同时存在时，and 语句的优先级要高于 or

7. python 中面向对象与类的使用

7.1 面向对象编程

面向对象编程是最有效的软件编写方法之一，在面向对象编程中，需要先编写具有同类特征的类，相当于软件编写的蓝图，定义了一个类，相当于定义了一个包含同类特征的设计图，在接下来的编程中，就可以使用这个设计图来创建实际的对象。

与面向对象相对立的一个编程思路为面向过程，如果用代码来建造一栋大楼的话，面向对象的思维则是将建造大楼的每一个环节设计好，定义好，在施工时就可以按照设计好的操作步骤重复实施。面向过程思维则是关注一栋大楼的实际创建过程，建造其他大楼时无法去复制这个过程。
