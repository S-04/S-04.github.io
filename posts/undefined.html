hexo+github（gitee）博客搭建

很早以前就听说过博客这种个人网站搭建的方式分享东西，以为都要需要比较多麻烦的事情，后来学习了这种简单的个人博客搭建的方式。基本也就是消耗点时间。下面就介绍下时如何搭建的吧。

  • 准备
  1. windows系统（linux也可以，这篇介绍的是Windows环境下的搭建）
  2. 安装Git
  3. 安装Node.js （安装时注意版本，如果是重新搭建，安装新版本后可能会报错。ps：安装时，请勾选Add to PATH选项）
  4. 注册github（gitee）账号
  • 搭建流程
  1. 设置user.name和user.email配置信息(注册的github用户名，邮箱)
  2. Git安装完成，鼠标右键打开Git Bash Here。

git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub注册邮箱"

3.生成ssh密钥文件

ssh-keygen -t rsa -C "你的GitHub注册邮箱"

然后直接三个回车即可，默认不需要设置密码 然后找到生成的.ssh文件夹中的id_rsa.pub密钥，将内容全部复制（文件路径：C:\Users\Administrator.ssh）

4.登陆github，创建Repository，如下图操作：

6.打开GitHub_Settings_keys页面，新建new SSH Key，Title为标题，随意填即可，将刚刚复制的id_rsa.pub内容复制进去，最后点击Add SSH key。

7.在Git Bash检测GitHub公钥设置是否成功

ssh git@github.com

  • 安装Hexo

    1.新建一个文件夹，用来存放博客文件，命名任意。（后续所有的博客文件和hexo框架都在里面），在文件夹中 -> Git Bash Here -> 使用npm命令安装Hexo。

npm install -g hexo-cli

ps：时间比较久。

hexo init Hexo

命令执行完成后会生成一个名为Hexo的文件夹 ，在文件夹中 -> Git Bash Here ，输入：

hexo new test
hexo g
hexo s

浏览器输入: http://localhost:4000/ 可以看到博客了。
